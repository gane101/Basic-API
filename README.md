This API is active on : http://nesh001.pythonanywhere.com/

To list all the animes in database, add this query - /anime/list

To find an anime in database, add this query - /anime/list/find and a json file containing "Name" of the said anime.

To add an anime to database, add this query - /anime/list/add and a json file containing "Name", "Synopsis" and "Rating" of the said anime.

To update some info about an anime in database, add this query - /anime/list/update and a json file containing "Name" and "Synopsis" or "Rating" of the said anime.

To delete an anime from database, add this query - /anime/list/delete and a json file containing "Name" of the anime.
